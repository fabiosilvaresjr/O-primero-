# Manipulação de Strings e Interação

> Projeto introdutório focado no uso de **Template Strings** e métodos de manipulação de texto em JavaScript.

Este script simples captura o nome do usuário através de um prompt nativo do navegador, realiza a contagem de caracteres da string fornecida e renderiza uma mensagem de boas-vindas personalizada diretamente no documento HTML.

## Tecnologias Utilizadas

- **HTML5** (Estrutura básica)
- **JavaScript** (Interação e lógica de strings)

## Funcionalidades

- [x] **Captura de Dados:** Uso do `window.prompt` para entrada de dados do usuário.
- [x] **Contagem de Caracteres:** Utilização da propriedade `.length` para analisar a string inserida.
- [x] **Renderização Dinâmica:** Uso de `document.write` (para fins didáticos) para escrever HTML via JavaScript.
- [x] **Interpolação de Strings:** Uso de sintaxe moderna para misturar texto e variáveis.

## Aprendizados e Destaques do Código

O foco deste estudo foi a sintaxe moderna do JavaScript (ES6+):

1. **Template Strings (`` ` ``):** Em vez de concatenar textos usando o sinal de mais (`"Olá " + nome`), utilizei Template Strings. Isso permite injetar variáveis diretamente no texto usando `${nome}`, deixando o código muito mais limpo e legível.
2. **Propriedades de String:** Aprendi que toda string em JavaScript tem propriedades nativas, como o `.length`, que retorna o tamanho exato do texto automaticamente.

## Como rodar o projeto

1. Clone este repositório.
2. Abra o arquivo `index.html` no seu navegador.
3. Responda à pergunta na caixa de diálogo e veja o resultado na tela.
